DEVSCORE — EVALUATION TEST CASES (ONE PER BUCKET)
===================================================
Assignment: "Simple Counter App"
Goal: Verify each of the 5 evaluation buckets independently.

Each section below contains:
  (A) TEACHER REFERENCE — the reference HTML/CSS/JS to paste when creating
      the assignment, plus the evalSpec JSON for that bucket's tests.
  (B) STUDENT SUBMISSION — a sample student answer to test with.
  (C) EXPECTED OUTCOME — what score/result you should see.

HOW TO RUN A TEST
-----------------
1. Teacher creates assignment via Teacher Dashboard, pastes reference code
   and the evalSpec JSON for THAT bucket only (leave other buckets empty []).
2. Student submits the corresponding student HTML/CSS/JS.
3. Wait for evaluation to complete, then check the result breakdown.
